    mixins

    comp_panel_margins

    @mixin comp_panel_margins($compPanelMarginsParam) { ... }

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $compPanelMarginsParam

    按顺时针四个外边距的值 类型：map 默认值：(top:0,right:0,bottom:0,left:0)

    		—none

    Example

    comp_panel_margins();默认，即外边距都为0
    comp_panel_margins((top: 10px,right: 10px,bottom: 20px,left: 5px));四个外边距都进行数值设置
    comp_panel_margins((top: 10px,bottom: 20px,left: 5px));只设置其中几个外边距，其余未设置默认为0

    Author

    • wangweihui
